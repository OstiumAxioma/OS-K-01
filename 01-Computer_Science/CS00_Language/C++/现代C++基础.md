## C++ 简介

1. **过程化程序设计**
    
    1. 按照计算步骤设计功能模块
    2. 程序通过调用模块来完成目标
    
    ```cpp
    输入调用指令 initial data
    经过预设指令 algarism 获得结果 target data
    ```
    
2. **面向对象程序设计**
    
    1. 从用户（对象）的角度出发，将一组过程和一组数据封装在一个类里
    2. 程序通过调用类来完成目标
3. **泛型程序设计**
    
    1. 编写的代码可以用于多种不同的数据类型
    2. 程序具有更高的复用性
    
    ```cpp
    F(X)=1+2+3+... #int
    F(X')=1.0+2.0+3.0+... #float
    ```
